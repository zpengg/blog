模版一 基本款

搜索条件不需要通过相邻元素决定。
不需要后处理。

初始条件： left = 0， right = len - 1；
结束条件： left > right;
搜左边： right = mid-1；
搜右边: left = mid+1;

mid = left + (right - left) / 2;

需要注意的是 搜索条件放在前面,边界的思路会清晰点。
